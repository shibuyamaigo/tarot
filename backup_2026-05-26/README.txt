# BITRIDERZ (Codename: RetroRide)

> The Retro Cycling Game  
> 画期的なサイクリングトレーニングアプリ

## 🎯 概要

レトロゲーム風UIで楽しめる、本格的なサイクリングトレーニングアプリ。
パワーメーターと連携し、軽量・高品質な体験を提供する。

## 🚀 ステータス

**Stealth Development - Coming Soon**

## 🏗 構成

このリポジトリは2つのプロジェクトで構成される。

### `/game/` - ゲーム本体
- HTML5 Canvas + Vanilla JavaScript
- 1MB以下を目指す軽量設計
- Web Bluetooth API でパワーメーター直結

### `/marketing/` - マーケサイト（暫定）
- Next.js 15 (App Router)
- TypeScript + Tailwind
- LP・課金導線
